# Harsh Gupta – Portfolio

Personal portfolio website showcasing projects, skills, and work experience.

**Live:** [itharsh.vercel.app](https://itharsh.vercel.app)

## Tech Stack

- React + Vite
- Tailwind CSS
- Framer Motion (animations)
- EmailJS (contact form)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
